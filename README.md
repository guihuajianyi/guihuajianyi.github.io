# 语录 — 名人经典语录展示平台

一个简洁优雅的语录展示网页，专注于展示单个人物的经典语录。使用DeepSeek AI生成。

## 功能特性

- 📝 **语录卡片展示** — 瀑布流网格布局，支持桌面/平板/手机三端自适应
- 🔍 **搜索过滤** — 支持按语录内容、作品名称、标签搜索
- 🏷️ **分类筛选** — 自动读取 `quotes.txt` 里的标签并生成筛选按钮
- ❤️ **收藏功能** — 收藏喜欢的语录，数据持久化到 localStorage
- 📋 **一键复制** — 点击复制语录及出处
- 🔍 **详情弹窗** — 点击卡片查看语录大图模式
- 📄 **分页加载** — 每页 24 条，支持翻页导航
- 📱 **全端适配** — 完美适配手机、平板、桌面端

## 使用方法

推荐使用本地服务器预览，这样页面可以正常读取 `quotes.txt`：
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

## 自定义

要更换作者和语录内容，只需修改：
- `index.html` — 作者信息（头像、姓名、简介、代表作品）
- `quotes.txt` — 语录正文和标签。每条语录可写成：
  ```text
  2026.6.6 14:37
  “语录内容”
  标签：自相矛盾、课堂
  ```
  标签行可省略，多个标签可用顿号、逗号、空格、斜杠或竖线分隔。
- `js/data.js` — 语录数据数组
- `css/style.css` — `:root` 中的 CSS 变量调整配色

# 语录 — 名人经典语录展示平台

一个简洁优雅的语录展示网页，专注于展示单个人物（史铁生）的经典语录。

## 功能特性

- 📝 **语录卡片展示** — 瀑布流网格布局，支持桌面/平板/手机三端自适应
- 🔍 **搜索过滤** — 支持按语录内容、作品名称、标签搜索
- 🏷️ **分类筛选** — 按生命、命运、爱情、孤独、希望、时间、苦难、哲理等标签分类
- ❤️ **收藏功能** — 收藏喜欢的语录，数据持久化到 localStorage
- 📋 **一键复制** — 点击复制语录及出处
- 🔍 **详情弹窗** — 点击卡片查看语录大图模式
- 📄 **分页加载** — 每页 12 条，支持翻页导航
- 📱 **全端适配** — 完美适配手机、平板、桌面端

## 项目结构

```
-sita-/
├── index.html        # 主页面
├── css/
│   └── style.css     # 响应式样式
├── js/
│   ├── data.js       # 语录数据
│   └── main.js       # 交互逻辑
└── README.md
```

## 使用方法

直接在浏览器中打开 `index.html` 即可预览。

或使用本地服务器：
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

## 自定义

要更换作者和语录内容，只需修改：
- `index.html` — 作者信息（头像、姓名、简介、代表作品）
- `js/data.js` — 语录数据数组
- `css/style.css` — `:root` 中的 CSS 变量调整配色
